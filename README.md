# Task Dawn v1.0.0 Alpha

> "Ignite your day with prioritized tasks in your inbox."

A retro-styled landing page for Task Dawn, a fictional task management utility built with 2003-era web design aesthetics.

## About

This is a static HTML website designed to look like shareware/utility software websites from the early 2000s. It features:

- Table-based layout (800px fixed width)
- Fieldsets and legends for section grouping
- Web-safe fonts (Tahoma, Arial, MS Sans Serif)
- Classic blue/purple link colors
- Visitor counter aesthetic
- "Best viewed in IE6" footer

## Deployment

This site is designed to be deployed via GitHub Pages:

1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and `/ (root)` folder
5. Your site will be live at `https://[username].github.io/[repo-name]/`
