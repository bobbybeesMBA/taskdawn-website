Step 1: Connect Google Account
🔒 Google OAuth2 Authentication

Task Dawn needs access to your Google Tasks. You'll need to create OAuth2 credentials in Google Cloud Console first.

📖 How to Get Your Google Client ID & Secret
1 Go to Google Cloud Console
https://console.cloud.google.com/
2 Create a New Project
Click project dropdown (top-left) → "New Project" → Name it Task Dawn
3 Enable Google Tasks API
Go to: APIs & Services → Library → Search Google Tasks API → Enable
4 Configure OAuth Consent Screen
APIs & Services → OAuth consent screen → Choose "External" → Fill in app name & email
5 Create OAuth Credentials
Credentials → + CREATE CREDENTIALS → OAuth client ID → Type: Web application
6 ⚠ CRITICAL: Add Redirect URI
Under "Authorized redirect URIs" add:
https://taskdawn.com/wizard/index.html
Must match exactly or OAuth will fail!
7 Copy Your Credentials
After clicking "Create", copy:
• Client ID: 123456789-abc...apps.googleusercontent.com
• Client Secret: GOCSPX-xxxxxx...

Step 2: iCloud Reminders Integration (Optional)
🍏 Connect iOS Reminders via IFTTT

Apple doesn't provide a public API for Reminders. We use IFTTT as a bridge to sync your iOS Reminders to a Google Tasks list that Task Dawn can read.

Don't use iCloud Reminders?
If you only use Google Tasks, skip this step.

📖 How to Set Up the IFTTT Bridge
1 Create a Google Tasks List
Open Google Tasks and create a new list named exactly:
Task Dawn Sync
2 Create an IFTTT Account
Go to https://ifttt.com/ and sign up (free)
3 Create a New Applet
Click "Create" → Click "If This" → Search for "iOS Reminders"
(Connect your iCloud account when prompted)
4 Choose Trigger
Select: "New reminder added to list"
Choose which Reminders list to sync
5 Set Up the Action
Click "Then That" → Search for "Google Tasks"
Select: "Create task"
6 ⚠ IMPORTANT: Configure Action
Task list: Task Dawn Sync
Title: {{Title}}
Notes: {{Notes}} (optional)
Due date: {{DueDate}} (optional)
7 Save and Enable
Click "Continue" → "Finish" → Make sure Applet is ON

✅ Test Your Setup:

1. On your iPhone, create a Reminder: IFTTT Test Task
2. Wait 1-5 minutes
3. Check "Task Dawn Sync" in Google Tasks
4. If it appears, you're all set! ✔

Pro Tip: "Hey Siri, remind me to..." will now flow into Task Dawn!

Step 3: Authorize GitHub
📁 GitHub Repository Setup

Task Dawn will create a private repository called task-dawn-engine to store your automation and credentials.

📖 How to Get Your GitHub Personal Access Token
1 Go to GitHub Token Settings
https://github.com/settings/tokens/new
2 Configure Token
Note: Task Dawn Automation
Expiration: 90 days or "No expiration"
3 ⚠ REQUIRED: Select These Scopes
☑ repo - Full control of private repositories
☑ workflow - Update GitHub Action workflows
Without these, Task Dawn cannot create your repository!
4 Generate and Copy
Click "Generate token" → Copy immediately (starts with ghp_)
Token won't be shown again!
