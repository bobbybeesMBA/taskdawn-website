Step 1: Connect Your Google Account
🔒 Google OAuth2 Authentication

To connect Task Dawn to your Google Tasks, you need to create OAuth credentials in Google Cloud Console, then authorize access below.

📖 How to Get Your Google Credentials
1 Go to console.cloud.google.com and create a new project
2 Enable the Google Tasks API (APIs & Services → Library)
3 Configure OAuth consent screen (External, add your email)
4 Create OAuth 2.0 Client ID (Credentials → Create → Web application)
5 Add redirect URI: https://taskdawn.com/wizard/index.html
6 Copy your Client ID and Client Secret below
